# Risk Analyzer

AI-powered financial risk analysis tool that extracts data from Excel and PDF financial reports, calculates key risk metrics, generates automated analysis reports, and provides an interactive chatbot for financial insights.

## Features
- Extract financial data using fuzzy matching
- Calculate ROA, Debt-to-Equity, Liquidity, and other ratios
- Generate trend charts and risk scores
- AI-generated financial risk report using Perplexity AI
- Interactive chatbot for financial Q&A
